## Data types

Data types used to define the type of data that a table column can contain. They are modified by Constraints.

![expr](/assets/images/sql-guide/type_name.svg)

### Mapping data types

| SQL Data Type | FeatureBase Data Type |
|---|---|
| int | int |
| bool | bool |
| timestamp | timestamp |
| decimal | decimal |
| string | keyed mutex |
| stringset | keyed set |
| id | mutex |
| idset | set |
