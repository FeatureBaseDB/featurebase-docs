## Step 2: create table

1. Click **Create table**.
2. Select the destination database.
3. Enter a table name and an optional description.

## Step 3: choose the primary key

{% include /cloud-table/cloud-summary-table-pk.md %}

1. Select the ID type.
2. Click **Create**.
