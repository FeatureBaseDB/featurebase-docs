## Deactivate a user

* Click **Configuration** > **Manage users**
* {% include /cloud-icons/icon-click-context-menu.md %} on the **Active** user > **Deactivate user**.

## Reactivate a user

* Click **Configuration** > **Manage users**
* {% include /cloud-icons/icon-click-context-menu.md %} on the **Deactivated** user > **Reactivate User**.
